Card Game

C++

Upon request for a project relating OOP and basic fundamentals of programming logic, I created a simple card game in which resembles the card game many of us know as "War". However, there is a twist to this version of the game. There are four additional cards within the deck, labeled as special cards. If a player draws said "special card", then they lose a point for that round. With a traditional scoring system in place, where the winner of the round (he/she who selects the most powerful card) gains two points for the round. The program is able to read in how many players intend to play within the current game (2-4 player capability). At the end of each round, the scoreboard is presented to show the current score of each player. First player to have 21 points or more, while winning by at least 2, wins the game.
